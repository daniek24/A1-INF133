ISC Search Engine

For CS-120 (Information Retrival), my group and I developed a Web Search Engine of the entire UCI ICS web database capable of retrieving all tested queries in under 250ms. Our querying system used inverted partial indexing of over 55,000 unique documents and 940,000 unique tokens and a ranked database retrieval system which combined tf-idf scores and a custom html tag scoring system. We deployed a front-end using Verscel and Typsecript converted from Python.
